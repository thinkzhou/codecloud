 Problem：1001 计算天数

Time Limit : 2000/1000ms (Java/Other)   Memory Limit : 131072/65536K (Java/Other)
Total Submission(s) : 3   Accepted Submission(s) : 0

  • Description

    十进制正整数都可写成二进制的形式，如20可写成10100，如果我们把该二进制数10100的后面的2个0分离，可将10100分成两部分（101，00），即10100=101*22。现在101的十进制表示是5，称为10100的“基本部分”，而00用0的个数2是它的“指数部分”。于是十进制正整数20可表示为（5，2）。
    你的任务是对一个十进制正整数n确定这样的表示。

  • Input

    输入有若干行，每行上有一个十进制整数n，1<=n<=65535。
    输入直到文件结束。

  • Output

    对输入中每一行上的一个十进制正整数n，在一行上先输出n的基本部分，再输出指数部分，两者之间留一个空格。

  • Sample Input

    12
    16
    131

  • Sample Output

    3 2
    1 4
    131 0
